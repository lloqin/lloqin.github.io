IMPORTANT: Presets are saved to the web browser you're using NOW. Import Presets

IMPORTANT: Existing presets will not be replaced. If an imported preset has the same name as one you already have, yours is kept and the incoming one is skipped. You may still want to export your local presets before importing, just to be safe.
